LBController¶

This page describes the class LBController

The class LBController manages a number of related LBBase objects (i.e. LayerBuf or LayerArrBuf).
Each of the LBBase objects must have the same number of layers and the same layer size.
A vector of L2List objects is used to control usage of the elements of the managed LBBase objects.
