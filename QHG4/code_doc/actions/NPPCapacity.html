NPPCapacity¶

This page describes the class NPPCapacity`

The class NPPCapacity is derived from Action and is used to calculate per-cell capacities based on NPP, sweetwater and coastel effects.

First an initial capacity is calculated from the local NPP using a ramp function. Then, if water is present in the cell, the cpacity is multiplied with a “water factor”. If the cell us close to the coast (i.e. if the entry in Geography’s Coastal array is greater than 0), the capacity is multiplied by a coastal factor.

The attributes regarding coastal and water only work if the geography has the appropriate data in its m_AdCoastal and m_adWater arrays. Uually these arrays are added to the gridfile by means of the tools VectorImport and Coastal.
