Evaluator¶

This page describes the class Evaluator

The pure virtual class Evaluator is derived from Action as well as Observer, and serves as a base class for classes that evaluate the environment.

The implementation of the Observer interface allows this action to be notified of events.

Code: Evaluator.h

This clss has no attributes.
