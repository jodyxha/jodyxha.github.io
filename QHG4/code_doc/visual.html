This page describes the directory visual

The directory visual contains some tools for the creation and modification of PNG images:

  • PNGImage: a wrapper for the png library with many useful auxiliary functions

  • AlphaComposer: alpha compositing images

  • TextRenderer: render text onto an image

  • XXXLookup: color lookup tables assigning an rgba color to an iunput value:
