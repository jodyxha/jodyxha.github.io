tut_OldAgeDiePop¶

This page describes the class tut_OldAgePop

The class tut_OldAgeDiePop is derived from SPopulation.

Here the agents age and die after reachin an age near the maximum age.

For this the actions

  • GetOld

  • OldAgeDeath

are required.

(The class tut_OldAgeDiePop is described in detail in the second part of the tutorial).

Code: tut_OldAgeDiePop.h, tut_OldAgeDiePop.cpp

Data types¶

The actions GetOld and OldAgeDeath expect the agents to have a float member m_fAge. Therefore, we extend the basic agent type defined in SPopulation by a float:

struct tut_OldAgeDieAgent : Agent {
    float m_fAge;
};
