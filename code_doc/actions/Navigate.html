Navigate¶

This page describes the class Navigate

The class Navigate is derived from Action and handles the sea crossing of agents. It manages the “bridges” between landmasses and calculates the crossing probabilities using a sampling distance \(d_0\), a crossing probability \(p_0\) for that distance and a decay \(D\):

\[p_C = A \cdot e^{D * d_0}\]

where \(A = p_0/e^{D*d_0}\)

In other words the probability for a crossing diminishes with growing distance between land masses. The values \(d_0\), \(p_0\) and \(D\) model the population’s ability to navigate the open sea.
